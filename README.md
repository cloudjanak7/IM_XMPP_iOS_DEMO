# 基于XMPPFramework实现的一个demo

## 目前实现的功能有<br>
    1.登录
    2.消息发送（文本，图片，语音）
    3.个人信息
    4.后台运行
    5.好友按组别分组
    6.历史聊天好友
    7.聊天界面优化（仿微信）

    3.忘了谁的汉字转拼音

    6.微信的图片库
